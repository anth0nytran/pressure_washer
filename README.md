# Pressure Washer Landing Page

A modern, responsive landing page built with Next.js, React, and Tailwind CSS.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

Edit `lib/demoDefaults.ts` to customize:
- Business name, city, phone
- Services offered
- Testimonials
- FAQs
- Colors and theme
- And more!

## Project Structure

- `app/page.tsx` - Main page that renders the TemplateHome component
- `templatehome.tsx` - Main landing page component
- `components/` - Supporting components (modals, sliders, etc.)
- `lib/demoDefaults.ts` - Configuration and default data
